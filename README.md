# Band Website

A modern, dark-themed website for showcasing your band's music, members, and upcoming shows.

## Features

- Responsive design
- Member carousel
- Music/services section
- Upcoming shows listing
- Contact information
- Smooth animations and transitions

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content with your band's information

## Customization

- Update band name in `index.html`
- Modify colors and styling in `styles.css`
- Add member images and information
- Update contact details

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Poppins Font

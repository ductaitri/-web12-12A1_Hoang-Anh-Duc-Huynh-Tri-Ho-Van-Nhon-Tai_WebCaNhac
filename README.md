# Website Ban Nhạc

Đây là trang web giới thiệu ban nhạc với giao diện hiện đại, hỗ trợ thiết bị di động, cung cấp thông tin về thành viên, nhạc, show diễn sắp tới và liên hệ.

## Chủ đề

**Ban nhạc** - Trang web giới thiệu thành viên, âm nhạc và sự kiện của một ban nhạc.

| Hoàng Anh Đức  | Lập trình      |
| Nhơn Tài       | Viết báo cáo   |

## Cách chạy dự án

1. **Clone** repository này về máy của bạn:
   ```bash
   git clone https://github.com/ductaitri/-web12-12A1_Hoang-Anh-Duc-Huynh-Tri-Ho-Van-Nhon-Tai_WebCaNhac.git
   ```
2. **Mở file** `index.html` trong thư mục gốc bằng trình duyệt của bạn.

3. Tuỳ chỉnh nội dung (`index.html`, `styles.css`) nếu muốn.
